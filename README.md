# BMW Sales Data Analysis Dashboard 

This repository contains a Tableau dashboard analyzing BMW sales performance from **2010 to 2024**. The dashboard highlights unit sales, revenue, transmission preferences, model popularity, color trends, fuel type distribution, and regional performance.

---

##  Key Highlights

### Sales Overview
- **Total Units Sold:** 253.38M  
- **Average Selling Price:** $75.0K  
- **Total Revenue:** $19,012.2G  

### Transmission Breakdown
- **Automatic:** 126.01M  
- **Manual:** 127.36M
